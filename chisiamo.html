Il nostro progetto

Siamo un gruppo di studenti di informatica appassionati di sviluppo web e programmazione. Questo sito è stato creato come progetto didattico per imparare:

  • HTML5 e CSS3 moderno
  • JavaScript e DOM manipulation
  • Responsive Web Design
  • Controllo versione con Git
  • Collaborazione su GitHub
